# CONTROLS

## Camera Controls
- WASD - Move camera
- Arrow Keys - Look around
- Space - Move up
- Left Shift - Move down

## Lighting Controls
- 1 - Toggle roof lights
- 4 - Toggle flashlight
- C - Cycle flashlight color (White → Red → Green → Blue)
- R - Toggle roof visibility
- Z - Generate new dungeon

## Other Controls
- ESC - Quit
